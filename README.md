Download
--------
Add jitpack repo in your root build.gradle at the end of repositories:

```groovy
allprojects {
    repositories {
        maven { url 'https://jitpack.io' }
    }
}
```
 Add the debouce dependency:
```groovy
dependencies {
    implementation 'com.github.rutvijkumarshah:debounce:1.0-beta'
}
```
